# 🚗 Car Price Prediction using Machine Learning

## 📌 Overview
This project predicts the selling price of used cars using a **Linear Regression** model. It demonstrates the complete machine learning workflow, including data preprocessing, feature encoding, model training, evaluation, and visualization.

## 🎯 Objectives
- Load and explore the car price dataset.
- Clean and preprocess the data.
- Convert categorical features into numerical values.
- Train a Linear Regression model.
- Predict car selling prices.
- Evaluate the model using performance metrics.

## 📂 Dataset
The dataset contains information about used cars, including:
- Car Name
- Year
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner
- Selling Price (Target Variable)

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 📊 Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Explore and clean the data.
4. Encode categorical variables using Label Encoding.
5. Split the dataset into training and testing sets.
6. Train a Linear Regression model.
7. Predict car selling prices.
8. Evaluate the model using R² Score and Mean Absolute Error (MAE).
9. Visualize Actual vs Predicted prices.

## 📈 Model Performance
- **Model:** Linear Regression
- **R² Score:** 0.8465
- **Mean Absolute Error (MAE):** 1.2147

## 📷 Output
The project includes:
- Dataset exploration
- Data preprocessing
- Model training
- Performance evaluation
- Scatter plot comparing Actual vs Predicted Selling Prices
